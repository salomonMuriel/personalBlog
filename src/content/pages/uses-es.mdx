## Sistema Operativo

[![Mi distro preferida.](@assets/posts/uses/logo-mint.png)](https://linuxmint.com/) Principalmente porque es basada en Debian, fácil de personalizar y excelente manejando múltiples pantallas con diferentes resoluciones y DPIs.

## Lenguajes de Programación, Frameworks y otros.

- **Lenguajes**: 🐍 Python, TypeScript y SQL.
- **Frameworks Backend**: FastAPI, Django y NodeJS.
- **Frameworks Frontend**: React + NextJS y Astro.
- **Motores de Base de Datos**: PostgreSQL es mi favorito, pero he tocado la mayoría de los motores principales. Odio Mongo 😠.
- **Cosas de Data Science**: Pandas, Polars, Redshift para warehousing, PySpark, Airflow, probablemente todos los productos de datos de AWS y GCP (lo siento Azure), Metabase, un montón de librerías de NLP, DVC, Tensorflow y Pytorch, scikit-learn + XGboost + LightGBM para modelos rápidos, Pycaret para modelos aún más rápidos, optuna para buenos modelos, y probablemente se me olvidan algunos. He pasado un montón de tiempo en Data Science 📊.
- **Proveedores Cloud**: GCP, AWS y Heroku (QEPD 💀). Actualmente estoy más familiarizado con las ofertas de AWS, en particular para despliegue web y datos.

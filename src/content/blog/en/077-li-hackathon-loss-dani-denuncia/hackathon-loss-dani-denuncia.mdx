You've got to share the losses too ⚰️ — the #IAHackathon got us.

[Rafael Sanabria](https://linkedin.com/in/rfsan) and I spent 3 hours on the bug from hell 👹 (seriously, I'm posting the code soon to see who can fix it). Not even pair programming + Claude Code + ChatGPT + Gemini 🫠.

With [Sebastian Martinez Hoyos](https://linkedin.com/in/sebasmartinezhoyos), the n8n flow for the main feature spent about 5 hours in limbo (mostly my fault 🙈) and at some point it looked more like an M.C. Escher painting than a workflow.

But it wasn't a waste of time. Most importantly — we learned A TON from actually doing:

1. Using Claude Code "for real" (advice: plan it out well and give it small pieces at a time, or before you know it, it's completely out of control)
2. n8n in turbo mode. Living through all its strengths and weaknesses. Incredibly powerful.
3. Canva Code for presentations. That was the best thing that came out of the whole thing — Sebas taught us.

After hours of unsolved bugs + eyes barely open but having had a great time and learned like crazy, we decided to close the laptops 💤

Dani Denuncia — the anti-crime bot — will come... just not this morning. We'll need to use the Claude Code subscription for the next version with a bit more time 👨‍💻. You can see the latest landing page we left here: [https://lnkd.in/eVtz5Eid](https://lnkd.in/eVtz5Eid)

The event organization was impeccable. Great food, snacks, water/coffee the whole time, super comfortable, good wifi, great vibes.

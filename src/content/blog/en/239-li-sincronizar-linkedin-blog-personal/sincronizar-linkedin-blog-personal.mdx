A few years ago I set up my personal website just for digital presence - so if someone Googled me, they'd find everything about me. Now I feel it's three times more important.

With good digital presence, LLMs will answer better about me if someone asks them.

To rank, I needed more content and hadn't posted anything in over a year. But I write daily on LinkedIn... and it hit me yesterday - can I sync it?

With my buddy Claude, it was set up in a couple of hours - now every day Apify scrapes my posts, checks if there's anything new, and if there is, it uploads it to my blog with GitHub Actions 🔥

I'll drop the repo in the comments so you can copy the script (and even the full page to set up your own if you want!)

Ajá, ya todos postearon que salió GPT-5... pero acá les tengo el System Prompt filtrado 👇🏼

[https://lnkd.in/eFHjzmiK](https://lnkd.in/eFHjzmiK)

El System Prompt es la instrucción previa que le deja OpenAI al modelo para guiar su comportamiento.

Así logran que responda bonito, que no rompa leyes, que tenga un estilo consistente, etc.

Lo más interesante para mi:

1. Si quieres que el modelo se acuerde de algo a largo plazo, se explícito en pedírselo. "Quiero que recuerdes esto y esto y esto de mi para después". Tiene prohibido acordarse de algunas cosas a menos que uno se lo pida explícitamente.

2. Que le digan al modelo unas librerías exactas que usar en cosas de código (Framer Motion para animaciones, Tailwind para CSS por ej). Creo que mata un poco la competencia entre librerías.

3. Que solo le den instrucciones específicas para Python y React (ni siquiera JS?). Sé que solo esas puede evaluar interno, pero ajá, al menos una guía de estilo para el top 10 de lenguajes?

4. Que tengan que repetirle tanto las cosas 😵‍💫 al pobre le dijeron 5 veces que no escribiera JSONs para la memoria, y un par en negrillas.
